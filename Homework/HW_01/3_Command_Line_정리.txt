3.1 커맨드 라인에 대한 간단한 요약

커맨드 라인이란? 커맨드 라인 인터페이스, cmd, CLI, 프롬프드, 콘솔, 터미널이라고 불리는 텍스트 기반 응용프로그램이다.
이는 파일보기, 처리, 조작이 가능하다.

커맨드 라인을 여는 방법은
1. 윈도우 : 시작 메뉴 -> 모든 프로그램 -> 보조프로그램 -> 명령 프롬프트
2. Mac OS X : 응용 프로그램 -> 유틸리티 -> 터미널
3. 리눅스 : 프로그램 -> 보조 프로그램 -> 터미널 // 시스템에 따라 다를 수 있다.

프롬프트(prompt)란? 컴퓨터가 입력을 받아들일 준비가 되었다는 것을 사용자에게 나타내는 신호 // \$, > 와 같은 기호

기본 명령어 : 각각의 운영체제에 따라 명령어들이 조금씩 다를 수 있다.
명령어로 현재 디렉터리 위치, 파일과 디렉터리 목록 보기, 디렉터리 변경, 새 디렉터리 생성, 삭제, 종료 등이 가능하다.

Window 기준 유용한 명령어

1. exit (창닫기)

2. cd (디렉터리 변경)

3. dir (디렉터리 목록 보기)

4. copy (파일 복사)

5. move (파일 이동)

6. mkdir (새 디렉터리 만들기)

7. del (디렉터리 혹은 파일 지우기)

3.2 커맨드라인 숙련도 평가

cmd라는 기능이 있다는 것은 알았지만 평소에 사용해 본적은 이번이 처음입니다. 모든 것이 처음이고 제시된 명령어나 제대로 된 사용법은 아직 잘 모르겠습니다. 하지만 예시와 예제에 따라 해보니 쉽고 생각보다 간단하였습니다.
