# 1. 게임의 소개
* **제목** : 드래곤 플라이트 (Dragon flights)
* **목적** : 드래곤 용사들이 내려오는 몬스터를 공격으로 처치하거나 피하면서 모험을 계속 진행해 나아가는 게임입니다.
* **방법** : 좌우 방향키를 이용하여 몬스터나, 몬스터들이 날리는 공격을 피하여 진행한다. (공격은 자동 공격이다.)
* **장르** : 종 스크롤 슈팅 게임

# 2. Gamestate (Scene)의 이름과 수
### Gamestate 5개
* Loading state
* Title state
* Menu state
* Play state 
* Game over state
-----
# 3. GameState (Scene)의 설명
* **Loading state**
  - 설명 : 게임 제목과 메인 일러스트가 뜨는 화면
  - 다른 state 이동
    + 일정 시간이 지나면 Title state로 이동한다.
* **Title state**
  - 설명 : 게임 시작과 게임 종료 화면
  - 구성 : 게임 시작, 종료 버튼
  - 이벤트 : 마우스 클릭
  - 다른 state 이동
    + 게임 시작 버튼을 마우스 클릭으로 누르면 Character select state로 이동한다.
    + 게임 종료 버튼을 마우스 클릭으로 누르면 게임을 종료한다.
* **Character select state**
  - 설명 : 캐릭터 선택 화면
  - 구성 : 3가지 캐릭터 프로필 이미지, 3가지 캐릭터 일러스터 이미지
  - 이벤트 : 좌우 방향키를 사용하여 캐릭터를 선택한 후 Enter 키를 입력하면 해당 캐릭터가 선택된다.
  - 다른 state 이동
    + 캐릭터를 선택 후 Enter 키를 입력하면 Play state로 이동한다.
* **Play state**
  - 설명 : 메인 게임 플레이 화면
  - 구성 : 플레이어 캐릭터, 아이템, 목숨(하트), 내려오는 몬스터, 점수, 배경
  - 이벤트 : 좌우 방향키를 사용하여 좌우로 이동한다.
  - 다른 state 이동
    + 목숨이 0이 되면 Game over state로 이동한다.
* **Game over state**
  - 설명 : 게임 오버시 나타나는 화면
  - 구성 : 점수(Score), 다시 시작, 게임 종료 버튼
  - 이벤트 : 마우스 클릭
  - 다른 state 이동
    + 다시 시작 버튼을 마우스 클릭으로 누르면 Character select state로 이동한다.
    + 게임 종료 버튼을 마우스 클릭으로 누르면 게임을 종료한다.

# 4. 필요한 기술

  - 캐릭터가 몬스터에 부딪히거나, 날아오는 투사체에 부딪히는 것을 확인하는 충돌 체크

  - 추가된 사운드의 소리 크기 조절
  - 좀 더 다양한 모듈

# Game state에 등장하는 Game object Class 정보
* **Player()**
  - handle_event() 함수를 사용하여 플레이어가 좌우로 움직인다.
  - 플레이어는 fire()함수에서 Bullet()를 불러와 총알을 발사한다.
  - 일반 몬스터와 충돌 하였을때 목숨이 1개씩 감소한다. (총 목숨은 3개)
* **Bullet()**
  - 총알은 플레이어의 위치보다 위쪽에서부터 그려져 나간다.
  - 파워는 50으로 설정하였다. (추후 변경 가능)
* **Enemy()**
  - enemy_gen 모듈을 이용하여 생성한다. (한번에 5마리 씩, 한 레벨이 5번 나오게 생성)
  - 아래로 이동한다.
  - 생명력은 Level * 100 씩 가진다.
* **Boss()**
  - 게임이 시작된 후 일정 시간이 흐른후 아래로 천천히 내려오며 나타난다.
  - 일정 Y위치까지 내려오면 좌우로 계속해서 움직인다.
* **Money()**
  - 플레이어가 일반 몬스터를 죽였을 때 생성
  - X는 좌측과 우측 둘중에 랜덤한 방향으로 움직이고, Y는 아래쪽으로 움직인다.
* **Score()**
  - 플레이어가 일반 몬스터를 죽였을 때 상승 (임시 : Level * 100점)
  - 플레이어가 money를 획득 하였을 때 상승 (임시 : 100점)
  - 플레이어가 보스 몬스터를 죽였을 때 상승 (임시 : 10,000점)
  - 스코어의 적절한 값은 추후에 완벽히 정할 예정
* **Background()**
  - 배경 이미지는 아래로 100의 속도로 지속해서 내려온다.

# 게임 진행 방법
* **Title_state**
  - 게임 시작 버튼을 누르면 캐릭터 선택 화면으로 이동한다.
  - 게임 종료 버튼을 누르면 게임이 종료된다.
* **character_selete_state**
  - 2가지 캐릭터를 선택 할 수 있다.
  - 방향키 ←, → 키를 사용하여 캐릭터를 선택하고 Enter키를 누르면 선택한 캐릭터로 게임이 진행된다.
* **Play_state**
  - 방향키 ←, → 키를 사용하여 캐릭터를 좌우로 이동할 수 있다.
  - space bar 를 누르면 파워샷 아이템이 발동 된다(총 9개)
  - L-Sift 키를 누르면 듀얼샷 아이템이 발동 된다.(총 10개까지 누적)
  - 몬스터를 피하고 보스 스테이지에서 아이템을 잘 사용하여 게임을 클리어 한다.
* **Game_over_state**
  - 플레이 스코어가 기록 된다.
  - 스코어는 거리 점수와 격파 점수를 합산한 값이다.
  - Enter 키를 누르면 Title 화면으로 돌아가 게임을 재시작 할 수 있다.
  - ESC 키를 누르면 게임이 종료 된다.
