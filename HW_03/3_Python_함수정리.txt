3) 함수 (function)

수학에서 함수란? 어떤 수식을 정의한 것을 의미한다.
반면 프로그래밍에서 함수란? 어떤 특정한 일을 처리하는 기능을 모아놓은 것을 의미한다.
이러한 프로그래밍에서 함수에서는 일반적으로 라이브러리, 모듈은 여러 개의 함수들로 구성되어 있다.
프로그래머는 프로그래밍을 하면서 자기 자신만의 함수를 만들어 이용하고 실행 시킬수 있다.
이러한 함수의 사용 시 코드가 길어질 시 확실하게 알아볼 수 있도록 함수의 이름을 그 함수의 기능에 맞추어 정확하게 표현하여 나타내어야 한다.

파이썬(python)에서 함수의 문법은
def 함수명 (매개변수) :
     < 수행할 문장>
       ...
으로 쓰여진다.

- def : 함수 정의임을 뜻한다.
- 함수명 : 말 그대로 함수의 이름이다.
- 매개변수 : 또는 인수라고 불러며 외부에서 전달되는 값을 의미한다.
- 수행할 문장 : 함수가 호출되어 사용될 때 실행되는 문장이다.

 
함수의 문법에서 가장 중요한 포인트는 수행할 문장을 들여쓰기 하여 코드를 작성하여야 한다.

함수는 코드에서 정의만 되어 있다고 그 자체로는 실행되지 않는다. 함수를 실행하기 위해서는 함수를 호출하여 사용하여야 한다. 그리고 함수를 호출 하기 위해서는 호출 코드 위에 함수 정의가 먼저 되어 있어야 호출하여 함수를 사용할 수 있다.

이러한 파이썬(python)에서 함수 사용의 장점은 여러 작업을 모아 하나로 처리할 수 있게 해주고, 함수 내에서 여러 개의 return값을 반환 할 수 있으며 매개변수(인자)의 타입에 따라 자동으로 연산 기능이 결정되는 점이다.
